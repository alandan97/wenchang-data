# 🎯 文创产品分析师 - 核心规范（重点标注版）

## 📍 项目定位
**本规范确保文创产品分析报告的标准化产出，保持视觉一致性和内容深度。**

---

## ⭐ 七大核心规范（必须遵守）

### 1️⃣ 标题规范 ⭐⭐⭐
```
❌ 禁止: "文创产品分析报告" 等 eyebrow 文字
✅ 必须: 直接显示产品名称

示例:
  主标题: 布达拉宫绿檀手串
  副标题: 布达拉宫文创 · 佛教文化 · 绿檀念珠
```

### 2️⃣ 图片轮播规范 ⭐⭐⭐
```
数量: 3-5张（必须）
来源: 优先官方渠道，其次百度图片
检查: 全部HTTP 200可访问
内容: 不重复，覆盖主图/细节/场景

图片类型:
  1. 产品主图（整体展示）
  2. 细节特写（材质工艺）
  3. 场景图（佩戴/使用）
  4. 系列图（产品矩阵）
  5. 包装图（品牌体验）

验证命令:
  curl -I https://example.com/image.png
```

### 3️⃣ KPI数据规范 ⭐⭐⭐
```
必须5个指标:
  1. 累计销量（带单位，如 10万+）
  2. 产品定价（带货币符号，如 ¥149）
  3. 累计营收（带单位，如 1490万）
  4. 转化率（百分比，如 4.5%）
  5. 复购率（百分比，如 32%）

数据来源分级:
  A级(95%+): 官方财报
  B级(80-95%): 电商平台
  C级(60-80%): 社交媒体
  D级(<60%): 估算值（标注*）
```

### 4️⃣ 产品信息规范 ⭐⭐⭐
```
必须5条:
  1. 出品方
  2. 上市时间
  3. 产品定位（高亮显示）
  4. 材质工艺
  5. 核心卖点

❌ 禁止: 任何标签出现在产品信息版块
```

### 5️⃣ 用户画像规范 ⭐⭐⭐
```
必须4维度:
  1. 年龄分布
  2. 性别比例
  3. 核心城市
  4. 收入水平/消费动机

标签要求:
  ✅ 总字数 ≤ 18字
  ✅ 放置于用户画像版块
  ✅ 示例: 潮玩、盲盒、IP运营、收藏、社交、送礼

字数检查:
  潮玩(2) + 盲盒(2) + IP运营(4) + 收藏(2) + 社交(2) + 送礼(2) = 14字 ✅
```

### 6️⃣ 竞品价格对比规范 ⭐⭐⭐
```
必须标注清晰:
  X轴标签: 本品\n品牌名 / 竞品\n竞品名
  图表标题: 竞品价格对比（品类名）
  竞品选择: 同品类/同价位/同目标用户

示例:
  本品\n布达拉宫  ¥71
  竞品\n故宫文创    ¥89
  竞品\n敦煌文创    ¥128
```

### 7️⃣ 挑战与机遇规范 ⭐⭐⭐
```
数量: 各6条（必须）

挑战维度:
  1. 市场风险（二手炒作、同质化）
  2. 产品风险（IP周期、成瘾争议）
  3. 运营风险（成本上升、盗版仿品）

机遇维度:
  1. 内容机遇（影视联名）
  2. 市场机遇（出海扩张）
  3. 技术机遇（虚拟潮玩/NFT）
  4. 场景机遇（主题乐园）
  5. 模式机遇（数字化藏品）
  6. 合作机遇（跨界联名）
```

## 🚀 快速使用流程

### Step 1: 信息采集（30分钟）
```
□ 截图产品页面（3-5张）
□ 记录基础信息（出品方/价格/销量）
□ 搜索竞品信息（2-3个）
□ 整理到Excel
```

### Step 2: 数据填充（20分钟）
```
□ 打开 wcfx-template.html
□ 替换所有 {{变量}}
□ 检查标签字数 ≤ 18
□ 验证图片链接有效
```

### Step 3: 质量检查（10分钟）
```
□ 对照七大核心规范检查
□ 预览页面效果
□ 部署到GitHub
```
