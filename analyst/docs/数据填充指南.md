# 文创分析师模板 - 数据填充指南

## 模板变量说明

### 基础信息

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{{PRODUCT_NAME}}` | 产品全称 | 泡泡玛特星星人怦然星动毛绒挂件盲盒 |
| `{{BRAND}}` | 品牌名 | 泡泡玛特国际集团 |
| `{{CATEGORY}}` | 品类 | 潮玩盲盒 |
| `{{SUBCATEGORY}}` | 细分品类 | 毛绒挂件盲盒 |

### KPI数据

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{{SALES_VOLUME}}` | 累计销量 | 10万+ |
| `{{PRICE}}` | 产品定价 | ¥149 |
| `{{REVENUE}}` | 累计营收 | 1490万 |
| `{{CONVERSION}}` | 转化率 | 4.5% |
| `{{REPURCHASE}}` | 复购率 | 32% |

### 产品信息

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{{MANUFACTURER}}` | 出品方 | 泡泡玛特国际集团 |
| `{{LAUNCH_DATE}}` | 上市时间 | 2025-01 |
| `{{POSITIONING}}` | 产品定位(高亮) | 毛绒挂件盲盒 |
| `{{MATERIAL}}` | 材质工艺 | 聚酯纤维/ABS/金属配件 |
| `{{SELLING_POINT}}` | 核心卖点 | 明星IP联名，盲盒惊喜体验 |

### 用户画像

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{{USER_AGE}}` | 年龄分布 | 18-30岁 |
| `{{USER_GENDER}}` | 性别比例 | 女性70% · 男性30% |
| `{{USER_CITY}}` | 核心城市 | 上海 · 北京 · 深圳 |
| `{{USER_INCOME}}` | 收入水平 | 中高收入 |
| `{{USER_TAGS}}` | 标签(≤18字) | `<span class="tag">潮玩</span>...` |

## 快速填充步骤

### 1. 准备图片
```bash
# 使用Chrome截图获取产品图片
chrome --headless --screenshot=image1.png --window-size=1400,900 "https://image.baidu.com/search/index?word=产品名"
```

### 2. 上传图片到GitHub
```bash
# 图片上传到 images/ 目录
# 获取raw链接: https://raw.githubusercontent.com/用户名/仓库/main/images/xxx.png
```

### 3. 填充模板
使用字符串替换或模板引擎填充所有 `{{变量}}`

### 4. 质量检查
- [ ] 标签总字数 ≤ 18字
- [ ] 挑战/机遇各6条
- [ ] 轮播图3-5张
- [ ] 无 eyebrow 文字
- [ ] 产品信息无标签

## 自动化脚本示例

```python
import re

def fill_template(template_path, data, output_path):
    with open(template_path, 'r', encoding='utf-8') as f:
        template = f.read()
    
    # 替换所有 {{变量}}
    for key, value in data.items():
        template = template.replace(f'{{{{{key}}}}}', str(value))
    
    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(template)
    
    print(f'已生成: {output_path}')

# 使用示例
data = { ... }  # 填充数据
fill_template('wcfx-template.html', data, 'wcfx-product-name.html')
```
