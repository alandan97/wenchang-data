# 文创产品目标信息分析逻辑说明

## 一、分析目标定义

### 1.1 什么是"目标信息"
目标信息 = 产品核心数据 + 用户特征 + 市场表现 + 竞争态势

### 1.2 分析目的
- 判断产品市场地位
- 识别用户真实需求
- 发现增长机会点
- 指导运营策略

---

## 二、信息采集清单

### 2.1 基础信息（5项）

| 信息项 | 采集来源 | 关键问题 |
|--------|----------|----------|
| 出品方 | 官网/工商信息 | 品牌实力如何？ |
| 上市时间 | 电商上架时间 | 处于什么生命周期？ |
| 产品定位 | 价格+宣传语 | 目标人群是谁？ |
| 材质工艺 | 产品详情页 | 品质感如何？ |
| 核心卖点 | 官方宣传+用户评价 | 差异化优势是什么？ |

### 2.2 核心数据（5项）

| 数据项 | 采集方法 | 分析价值 |
|--------|----------|----------|
| 累计销量 | 官方公布/电商推算 | 市场接受度 |
| 产品定价 | 官方售价 | 价格带位置 |
| 累计营收 | 销量×定价 | 商业价值 |
| 转化率 | 购买/访问 | 产品吸引力 |
| 复购率 | 重复购买比例 | 用户忠诚度 |

**数据采集优先级：**
1. 官方财报/发布会（最准）
2. 电商平台数据（次之）
3. 行业报告估算（参考）

### 2.3 用户画像（4维度）

```
用户画像 = 人口统计 + 行为特征 + 心理动机 + 社交属性

人口统计：年龄/性别/地域/收入
    ↓
行为特征：购买频次/渠道偏好/使用场景
    ↓
心理动机：为什么买？情感需求/功能需求
    ↓
社交属性：分享意愿/社群归属/影响力
```

**标签提炼原则：**
- 总字数 ≤ 18字
- 覆盖产品属性+用户行为
- 示例：潮玩、盲盒、IP运营、收藏、社交、送礼

### 2.4 销售渠道（4个核心）

| 渠道 | 关键指标 | 分析重点 |
|------|----------|----------|
| 抖音 | 直播转化率 | 内容种草能力 |
| 天猫 | 搜索排名 | 品牌心智占领 |
| 小红书 | 笔记互动率 | 社交传播力 |
| 线下店 | 坪效/体验 | 场景价值 |

### 2.5 图片要求

**数量要求:**
- 最少3张，最多5张
- 当前案例: 4张 ✅

**图片来源优先级:**
1. **官方渠道** (最佳): 品牌官网/天猫旗舰店/官方社交媒体
2. **电商平台** (次之): 天猫/京东/抖音商品页
3. **搜索引擎** (可用): 百度图片/谷歌图片
4. **用户晒单** (补充): 小红书/微博用户分享

**图片内容要求:**

|------|----------|------|

| 3 | 场景图 | 佩戴/使用 |

**图片检查清单:**
- [ ] 数量 3-5张
- [ ] 全部可访问（HTTP 200）
- [ ] 内容不重复
- [ ] 清晰度足够（>500KB建议）
- [ ] 与产品相关

**图片验证命令:**

**挑战识别：**
- 市场风险：二手炒作、同质化
- 产品风险：IP周期短、成瘾争议

**机遇挖掘：**

## 三、分析逻辑流程

### 3.1 四步分析法

```
Step 1: 信息采集
    ├── 确定信息源（官方/电商/媒体/用户）
    ├── 交叉验证（同一数据3个来源）
    └── 缺失标注（估算值标注*）

Step 2: 数据清洗
    ├── 去重（同一数据只保留最新）
    ├── 标准化（统一单位/格式）
    └── 分级（A/B/C/D级可信度）

Step 3: 对比分析
    ├── 时间对比（环比/同比趋势）
    ├── 竞品对比（差距/优势）
    └── 行业对比（均值/标杆）

Step 4: 洞察输出
    ├── 现状判断（健康度评估）
    ├── 问题诊断（核心矛盾）
    └── 策略建议（ actionable ）
```

### 3.2 关键分析模型

**产品健康度评估：**
```
健康度 = 销量增速 × 复购率 × 渠道覆盖 × 品牌认知

- 优秀：健康度 > 80分
- 良好：60-80分
- 一般：40-60分
- 风险：< 40分
```

**用户价值评估：**
```
用户价值 = 购买频次 × 客单价 × 推荐率 × 生命周期

高价值用户：年消费 > 1000元，推荐 > 3人
中价值用户：年消费 500-1000元，有推荐
低价值用户：年消费 < 500元，无推荐
```

**渠道效率评估：**
```
渠道效率 = 流量 × 转化率 × 客单价 / 运营成本

高效渠道：ROI > 3
中效渠道：ROI 1-3
低效渠道：ROI < 1
```

## 四、信息呈现规范

### 4.1 数据呈现原则

**有数说数：**
- ❌ "销量很好"
- ✅ "月销量10万+，同比增长150%"

**有对比：**
- ❌ "价格149元"
- ✅ "定价149元，高于竞品A 15%，低于竞品B 8%"

**有结论：**
- ❌ "用户是年轻人"
- ✅ "核心用户18-30岁女性（70%），月可支配收入8000+"

### 4.2 标签提炼方法

**原始标签 → 精简标签：**
```
潮玩文化(4) + 盲盒经济(4) + IP运营(4) + 
收藏(2) + 社交(2) + 收藏爱好(4) + 
盲盒惊喜(4) + 社交货币(4) + IP情感(3) + 送礼(2)
= 33字 ❌

精简后：
潮玩(2) + 盲盒(2) + IP运营(4) + 
收藏(2) + 社交(2) + 送礼(2)
= 14字 ✅
```

### 4.3 挑战机遇撰写规范

**结构：问题 + 影响 + 应对**

**挑战示例：**
```
二手市场炒作
├── 问题：隐藏款溢价300%+，普通款滞销
├── 影响：品牌定价权削弱，用户体验分化
└── 应对：官方二手平台+动态定价机制
```

**机遇示例：**
```
出海东南亚
├── 市场：东南亚潮玩市场年增40%
├── 优势：文化亲近+价格优势+IP认知
└── 策略：TikTok种草+本地代理+限定款
```

## 七、常见问题

**Q1: 数据找不到怎么办？**
A: 用行业均值估算，标注"估算值*"，或定性描述。

**Q2: 竞品怎么选？**
A: 同价格带+同品类+同目标用户，选头部2-3个。

**Q3: 标签怎么提炼？**
A: 先穷举所有可能标签，再按重要性排序，取前6个，确保总字数≤18。

**Q4: 挑战机遇写不出6条？**
A: 从市场/产品/运营/政策/技术/社会六个维度各找1条。
